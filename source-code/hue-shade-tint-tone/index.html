The Color Wheel

Hue, shade, tint and tone:

Simply put, tints, tones and shades are variations of hues, or colors, on the color wheel.

You can create shades, tints and tones of a color by adding black, grey and white to a base hue.

  • A tint is a hue to which white has been added, lightening the color. This can make a color less intense, and is useful when balancing more vivid color combinations.
    For example: red + white = pink.
  • A shade is a hue to which black has been added, darkening the color. This creates a deeper, richer color. Shades can be quite dramatic and can be overpowering.
    For example: red + black = burgundy.

A tone is a color to which black and white (or grey) have been added. This darkens the original hue while making the color appear more subtle and less intense. Tones are less likely to look pastel, and can reveal complexities not apparent in the base color.
