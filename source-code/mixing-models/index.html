The Color Wheel

There are two Mixing Models:

  1. RGB: the additive color mixing model
    • The RGB (red, green and blue) color wheel is designed for online use , as it refers to mixing light – like on a computer, TVs, screens and projectors. Mixed together they create pure white light, the more light you add, the brighter the color mix becomes. If you mix all three colors of light, you get pure, white light.
    • The RYB (red, yellow, blue) color wheel: is typically used by artists, as it helps with combining paint colors.
  2. CMYK: the subtractive color mixing model
    • Any color you see on a physical surface (paper, signage, packaging, etc.) uses the subtractive colors - like printers. The printers use the subtractive colors in order to get accurate color reproduction that can only be achieved by using CMYK and using RGB will only result in inaccurate colors.
